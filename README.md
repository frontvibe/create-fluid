# create-fluid

## Scaffolding for Fluid

**With NPM:**

```bash
npm create fluid@latest
```

**With Yarn:**

```bash
yarn create fluid
```

**With PNPM:**

```bash
pnpm create fluid
```

### CLI Flags

May be provided in place of prompts

| Name                         | Description                    |
| :--------------------------- | :----------------------------- |
| `--help` (`-h`)              | Display available flags.       |
| `--install` / `--no-install` | Install dependencies (or not). |
| `--git` / `--no-git`         | Initialize git repo (or not).  |
